# Antigravity CLI Add-on

Add-on for Home Assistant that provides a terminal with an AI Agent capable of controlling the domotics via the Supervisor API and MCP.

## Features

- **Persistent Sessions**: Powered by `dtach` and `script`, your terminal session remains active in the background. You can switch between PC, tablet, and mobile without losing your command history or running tasks.
- **Flawless Mobile Experience**: Custom-built momentum physics engine provides pixel-perfect, native-feeling touch scrolling on mobile devices, bypassing the limitations of standard web terminals.
- **Clean UI**: A custom HTML/JS wrapper around `ttyd` provides a distraction-free interface (no resize overlays or status bars).
- **Direct Image Uploads**: Built-in functionality to upload images directly to Home Assistant's `/share` directory. The path is automatically copied to your clipboard for quick pasting into the terminal.
- **Home Assistant Integration**: Automatically configures the MCP bridge with your Supervisor Token, giving the AI agent deep access to your domotics environment.
- **Custom Integration**: Appears neatly in your Home Assistant sidebar with the Antigravity logo (`mdi:triangle-outline`).
